CSS Classes Reference
/* Containers */
.ux-container       /* Max-width 1280px, centered */
.ux-container-fluid /* Full width */

/* Row */
.ux-row             /* 12-column grid container */

/* Base Columns (mobile-first) */
.ux-col-1 to .ux-col-12    /* Default (all screens) */

/* Responsive Breakpoints */
.ux-col-md-1 to .ux-col-md-12  /* Medium (768px+) */
.ux-col-lg-1 to .ux-col-lg-12  /* Large (1024px+) */
.ux-col-xl-1 to .ux-col-xl-12  /* XL (1280px+) */

/* Breakpoint Values */
md:  768px   /* Tablets */
lg:  1024px  /* Desktop */
xl:  1280px  /* Large desktop */

HTML + JavaScript

// JavaScript API

AJAX Tabs (Dynamic Loading)

HTML - Static
<div class="ux-tabs">
  <div class="ux-tabs-list">
    <button class="ux-tab is-active">Tab 1</button>
    <button class="ux-tab">Tab 2</button>
  </div>
  <div class="ux-tabs-panels">
    <div class="ux-tab-panel is-active">Content 1</div>
    <div class="ux-tab-panel">Content 2</div>
  </div>
</div>
HTML - AJAX
<div class="ux-tabs">
  <div class="ux-tabs-list">
    <button class="ux-tab is-active" data-url="/api/tab1.json">Tab 1</button>
    <button class="ux-tab" data-url="/api/tab2.json">Tab 2</button>
    <button class="ux-tab" data-url="/api/tab3.json" data-cache="false">No Cache</button>
  </div>
  <div class="ux-tabs-panels">
    <div class="ux-tab-panel is-active"></div>
    <div class="ux-tab-panel"></div>
    <div class="ux-tab-panel"></div>
  </div>
</div>
Data Attributes & API
/* Data Attributes (on .ux-tab) */
data-url="..."         /* URL to fetch content from */
data-cache="false"    /* Disable caching, refetch on click */

/* Events */
ux-tabs:change   /* Fired when tab changes { index, tab, panel } */
ux-tabs:loaded   /* Fired when AJAX content loads { index, content } */
ux-tabs:error    /* Fired on AJAX error { index, error } */

Alerts

Contextual feedback messages for user actions.

Information This is an informational alert using ux-alert-info.
Success Your changes have been saved using ux-alert-success.
Warning Please review your input using ux-alert-warning.
Error Something went wrong using ux-alert-error.
HTML
<div class="ux-alert ux-alert-info">
  <svg class="ux-alert-icon">...</svg>
  <div class="ux-alert-content">
    <strong class="ux-alert-title">Title</strong>
    Message text here
  </div>
</div>
CSS Classes Reference
/* Base Classes */
.ux-alert          /* Container - required */
.ux-alert-icon     /* Icon wrapper */
.ux-alert-content  /* Content wrapper */
.ux-alert-title    /* Bold title text */

/* Variant Classes (add to .ux-alert) */
.ux-alert-info     /* Blue - informational */
.ux-alert-success  /* Green - success action */
.ux-alert-warning  /* Yellow - warning/caution */
.ux-alert-error    /* Red - error/danger */
